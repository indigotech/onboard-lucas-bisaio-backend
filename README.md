# AMAZING DATABASE

## Project Description

This project have the propose to pratice and learn more about the backend's world.

## Environment and Tools

- Node.js v12.17.9
- Visual Studio Code v1.55.2
- Git v2.24.3
- TypeORM v0.2.32
- TypeScript v4.2.4
- GraphQL v15.5.0
- Mocha v8.3.2

## Steps to Run

- Install dependencies: `npm install`
- Deploy Database: `npm docker` or `docker-compose up -d`
- Build: `npm run build`
- Run: `npm start`

## Steps to Debug

- Lint: `npm run lint`
- Develop: `npm run dev`
- Tests: `npm run test`
